# Live Demos and Examples

Given the nature of the project, I have decided to split the demos into three separate components, each showcasing a different aspect of the project:

1. **Hugging Face Spaces**: This component is responsible for hosting a 124 M model (based on `miniGPT` architecture) on HuggingFace. It also serves as the backend for the API and a live interactive demo. Access it [here](https://huggingface.co/spaces/ableian/scgpt).

2. **Live website**: This component is actually a live website that demonstrates the capabilities of my trained model as a web app separate from Hugging Face, designed to be more user-friendly (as I said, my UI). It is hosted on vercel, access it [here](https://stupidcookgpt.vercel.app).

3. **Colab notebook**: This component is a jupyter notebook hosted on Google Colab which allows the users to see and use the training process as well as a summary of the code and the project. Access it [here](https://colab.research.google.com/drive/1n9l8sXo2m7j3a5e9v6z8x9y0w1z2a3b?usp=sharing).
